**安仔** @geekshellio 2026-02-22

终于有人把 OpenClaw Skills 整理清楚了。

这份列表从 5705 个技能里筛出 3002 个高质量的，过滤掉了垃圾账号、恶意代码、重复内容，分成 30+ 个类别。覆盖编程、自动化、智能家居、安全防护……工作量不小。

我在这个基础上，再帮新手筛一遍，挑几个日常最实用的出来，方便大家先试试手：

【安全防护类（必装）】

• skill-vetting - 装任何 Skill 之前先用它检查安全性

• backup - 备份你的 OpenClaw 配置，别折腾坏了

【编程开发类】

• coding-agent - 运行 Codex、Claude Code、OpenCode 等编程助手

• github - GitHub 操作（issues、PRs、CI）

• docker-essentials - Docker 容器管理

【日常生产力】

• apple-reminders - 管理 Apple 提醒事项

• weather - 天气查询

• imap-smtp-email - 邮件收发（支持 Gmail、Outlook、163 等）

【内容创作】

• anzai-image-gen - AI 图像生成（OpenAI、Google、DashScope）

• anzai-writing-style - 模仿特定写作风格

• mistral-ocr - PDF/图片转 Markdown

【浏览器自动化】

• browser - 浏览器控制和自动化

【通讯工具】

• discord - Discord 操作

这些都是本地优先、隐私安全的。

安装方式：

npx clawhub@latest install <skill-slug>

建议先装 5-10 个，跑通了再慢慢扩展。

从 0 到 1，比从 0 到 3002 应该会容易多了。
