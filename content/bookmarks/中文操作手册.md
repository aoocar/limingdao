## 快速安装

使用安装脚本一键安装 OpenClaw：

```
curl -fsSL https://openclaw.ai/install | bash
```

或使用包管理器安装：

```
# npm (需要 Node.js >= 22)
npm install -g openclaw

# Homebrew (macOS)
brew install openclaw

# pnpm
pnpm add -g openclaw
```

安装完成后，运行 `openclaw` 命令启动配置向导。 [查看详细安装指南 →](https://openclaw.aialiang.com/tutorial/install-index.html)
